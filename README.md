# MA4034 - Time Series and Stochastic Process

Course notes and study materials for MA4034: Time Series and Stochastic Process.

## Contents

This repository contains detailed LaTeX lecture notes covering key topics in time series analysis and stochastic processes:

- **Markov Chains**: Irreducibility, recurrence, transience, and limiting distributions
- **Random Walks**: Multi-dimensional random walks and return probabilities
- **Branching Processes**: Extinction probability and growth analysis
- **Eigenvalue Methods**: Finding limiting distributions using eigenvalue decomposition
- **Stationary Distributions**: Calculations and applications
- **Transition Matrices**: N-step transition matrices and their properties

## Topics Covered

- Concept clarifications for Markov chains
- Branching process analysis using generating functions
- ON/OFF Markov chain eigenvalue analysis
- Fair gambler's ruin problem
- Series criterion for recurrence
- Eigenvalue and limiting distribution methods

All lecture notes are provided in LaTeX format (.tex files).
